## 📖 About The Project

**Priska Cafe** is a professional website built for Priska Cafe — a destination for those who seek an exceptional coffee experience. The site features an elegant, user-friendly interface that reflects the cafe's refined identity and warm atmosphere.

The website allows customers to:
- Discover the story behind Priska Cafe and what makes it special
- Browse the full menu of drinks and food
- Reserve a table with ease
- Explore a photo gallery showcasing the cafe's ambiance
- Get in touch with the team directly

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌐 Bilingual | Supports Arabic and English |
| 📱 Responsive | Works on all devices and screen sizes |
| 🖼️ Photo Gallery | Showcase the cafe's atmosphere with high-quality images |
| 📋 Menu Display | Full browsable menu of drinks and food |
| 📅 Table Booking | Integrated reservation system |
| ⭐ Customer Reviews | Dedicated section for customer testimonials |
